Payment Flow

End-to-End Process

01

Connect Wallet

Connect any supported Solana wallet: Phantom, Solflare, Binance Web3, OKX, Trust Wallet, Bybit, or Gate.io. SOLQ never touches private keys. All signing happens in the external wallet app.

02

Scan QRIS Code

Point camera at any standard QRIS code — GoPay, Dana, BCA, or any merchant. SOLQ parses the EMVCo payload and detects Dynamic vs Static QRIS automatically via CRC-16/CCITT-FALSE validation.

03

Review Live Quote

Multi-source oracle: CoinGecko for SOL/USDC price, Jupiter V2 for best swap route, ExchangeRate API for IDR. Circuit Breaker blocks at >2% deviation to protect against manipulation.

04

Authorize & Sign

Tap "Launch External Wallet". Your wallet app signs the transaction. SOLQ only requests authorization — it never executes without your explicit cryptographic signature.

05

On-Chain Execution

Jupiter ExactOut swap: SOL/USDC → IDRX. Platform fee of 1.0% (platformFeeBps: 100) is embedded atomically in the swap and routed to Treasury. Blockchain confirmation: 3–10 seconds.

06

Settlement & Receipt

Setelah transaksi on-chain sukses, aplikasi menampilkan struk digital dengan detail ID Merchant dan Transaction Hash. User bisa langsung mengklik link untuk melihat bukti transaksi di Solana Explorer secara nyata.

Core Implementation

Technical Architecture

Universal Wallet Connectivity

Mobile: Universal Links + Mobile Wallet Adapter (MWA) — triggers Phantom, Solflare, Jupiter App. Web: auto-detects Phantom, Solflare, Backpack, OKX Wallet, MetaMask Solana Snap. Deep Bridging fallback for direct transaction building.

No Halu: Hardened Price Discovery

Oracle Multi-Source: CoinGecko SOL/USDC, Jupiter rute swap terbaik, dan kurs IDR dari ExchangeRate API. Circuit Breaker (2%): Jika swap menyimpang > 2% dari harga pasar global, transaksi DIBLOKIR OTOMATIS mencegah kerugian user.

Jupiter ExactOut Economic Engine

System reverse-engineers exactly how much SOL is required to produce the target IDR amount. Platform fee 1.0% is atomically embedded via platformFeeBps: 100 — revenue routes on-chain to Treasury wallet ETcQvs...P2m in the same atomic transaction.

QRIS Intelligence — Mata Pinter

EMVCo-compliant TLV parser. Tag 54 = Dynamic QRIS (amount locked). Missing Tag 54 = Static QRIS (manual input mode). Tag 59 extracts merchant name. Tags 26/27 extract merchant PAN for settlement routing. CRC-16/CCITT-FALSE validates data integrity.

Immutable State Machine

CREATED → AUTHORIZATION_REQUESTED → AUTHORIZED → AWAITING_SETTLEMENT → COMPLETED. No state skipping. No ambiguous transitions. Prevents UI-bypass attacks — "Settlement Completed" screen only renders after on-chain finality is confirmed by RPC polling.

Chain-of-Truth Verification

Double-check: Client-side Flutter RPC Polling + Server-side reconstruction. Backend verifies correct mint (IDRX/USDC, not fakes), correct treasury destination address, and finalized status on Solana Mainnet-Beta before marking success.

Stateless Orchestration at Scale

Memory-First Intent Store with auto-expiring TTL (5–10 min) — no SQL bottleneck on the critical payment path. ReconciliationWorker runs every 60 seconds to sweep stuck transactions and prevent memory leaks. Supports 30,000+ concurrent transactions.

Congestion Control & Priority Fees

Dynamic Compute Budget via computeUnitPriceMicroLamports ensures transactions are prioritized during Solana network congestion. Jito Bundle-ready architecture for guaranteed first-block inclusion on high-volume periods.

Non-Custodial by Design

SOLQ does not store balances, does not custody funds, does not act as an e-wallet, and does not issue QR codes. Read-Only App / Write-Only Wallet principle: every transaction is a Proposal only — execution requires the user's own cryptographic signature.
